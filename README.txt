The seven folders contain the MATLAB codes of Holimap for seven different networks.


-------------------------------------------------------------------------------
Folder 1: Autoregulation (network shown in Fig. 2(a))

"main.m" is the main program.
"master.m" contains the FSP algrithm.
"master_2HM.m" contains the 2-HM algrithm.
"master_4HM.m" contains the 4-HM algrithm.


-------------------------------------------------------------------------------
Folder 2: Repressilator (network shown in Fig. 4(a))

"main.m" is the main program.
"master_2HM.m" contains the 2-HM algrithm.


-------------------------------------------------------------------------------
Folder 3: Random Network (network shown in Fig. 5(c))

"main.m" is the main program.
"master_2HM.m" contains the 2-HM algrithm.


-------------------------------------------------------------------------------
Folder 4: Protein Sequestration (network shown in Fig. 6(a))

"main.m" is the main program.
"master_3HM.m" contains the 3-HM algrithm.


-------------------------------------------------------------------------------
Folder 5: Protein Phosphorylation (network shown in Fig. 6(b))

"main.m" is the main program.
"master_3HM.m" contains the 3-HM algrithm.


-------------------------------------------------------------------------------
Folder 6: mRNA Degradation Control (network shown in Fig. 6(d))

"main.m" is the main program.
"master_3HM.m" contains the 3-HM algrithm.


-------------------------------------------------------------------------------
Folder 7: microRNA-mRNA Interactions (network shown in Fig. 6(e))

"main.m" is the main program.
"master_3HM.m" contains the 3-HM algrithm.
